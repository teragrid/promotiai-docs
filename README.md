# promotiai-docs

Public documentation assets for [promotiai.com](https://promotiai.com) — content that lives outside the main
application repo so it can be published, forked, and contributed to independently.

## Skill Templates

[`skills/`](skills/) — eleven persona/vertical [skill](https://modelcontextprotocol.io) templates for running
PromotiAI from an AI agent (Claude, or any MCP client) via the
[PromotiAI MCP server](https://promotiai.com/docs/guides/skills-and-mcp) instead of the dashboard UI. Each
template mirrors a vertical from PromotiAI's own Vertical Wizard — pick the one closest to your business, copy
it into your agent's skills directory, and fill in the placeholders.

Start here: [Skills & MCP: Automate PromotiAI with AI Agents](https://promotiai.com/docs/guides/skills-and-mcp)
